# ADR-001: Unattended authority and honest terminal milestone

Status: accepted  
Date: 2026-07-11

The initiating instruction explicitly authorises public repository creation,
source acquisition from public interfaces, publication, releases, Pages,
protected-main configuration and an Explorer registry pull request. It does not
authorise authenticated sources, external shard hosting, paid API fan-out or a
human-participant study.

Consequently deterministic work and public-source acquisition proceed. No
external billable model calls are made. The honest release milestone is the
machine release candidate; human effectiveness and “UI of choice” remain
`not_yet_testable` until a separately authorised study completes.
